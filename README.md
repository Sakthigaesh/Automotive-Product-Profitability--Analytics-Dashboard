# Automotive Product Profitability Analytics Dashboard

This project presents an Automotive Product Profitability Analytics Dashboard built using Power BI. The dashboard analyzes car sales data to understand product profitability, pricing trends, and market performance.

## Project Objective
The objective of this project is to analyze automotive sales data and identify profitable products, customer demand, and pricing trends to support better business decisions.

## Tools & Technologies
Power BI  
Data Modeling  
DAX  
ETL Process  
CSV Dataset  

## Dataset
The dataset contains automotive product sales information including product categories, pricing, sales performance, and profitability.

## Dashboard Pages

Page 1: Product Feacture Demand
Page 2: Pricing Profit Analysis
Page 3: Market Trend Patterns 
Page 4: Corelation and Pattern Analysis
